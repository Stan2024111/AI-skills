Фокус: личная эффективность

Ключевые навыки

3 пункта

1 Работаю с основными нейросетями

Уверенно использую GPT, DeepSeek, Perplexity, Gemini под разные задачи: анализ, генерация, поиск, резюме, структурирование.

2 Создаю разные виды промтов

Промты для роли/контекста, пошаговые инструкции, контроль качества (критерии), рефрейминг, итеративные цепочки.

3 Применяю нейросети для личной эффективности

Планирование, подготовка документов, ускорение рутины, конспекты, структурирование мыслей, чек-листы, шаблоны.

Акцент: превращаю запрос в понятный результат — структурированный текст, план действий или набор решений с критериями качества.

Примеры кейсов

блоки
Кейс: Быстрое принятие решений 30–60 мин

Из хаоса входных данных → в краткий план и список приоритетов.

  • Сбор требований и контекст
  • Структура: цель → ограничения → варианты → риски
  • Итог: план действий и критерии проверки
Кейс: Документы и шаблоны качество

Ускорение подготовки текстов без потери смысла.

  • Черновик → уточнение → финальная версия
  • Единый стиль и визуальная логика
  • Списки, чек-листы, регламенты
